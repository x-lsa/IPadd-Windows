### IP address for Ethernet
```
C:\Windows\System32>ipadd help

--- IPADD Utility Help ---
  ipadd list             : Dashboard showing IPs, VLAN ID, and Hardware.
  ipadd clear            : Force-reset to DHCP and remove VLAN tags.
  ipadd vlan [ID]        : Force a VLAN tag (e.g., ipadd vlan 100).
  ipadd vlan clear       : Set VLAN back to 0 (Untagged).
  ipadd [IP]/[CIDR]      : Add/Set a static IP (e.g., 10.0.0.5/24).
  ipadd help             : Show this menu.
```
